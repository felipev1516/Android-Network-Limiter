<h2>
  Description:
</h2>
<p>
  A simple PowerShell Script that utilizes its .NET Windows Forms library and focuses on controlling Internet Bandwidth by using Android Debug Bridge (ADB). This will be used for network testing for android application without having to switch tabs to the developer settings in the device. 
</p>
<h2>
  Installation:
</h2>
<p>
  1.) Make sure the Android device is on Android 13 or newer and developer mode is turned on. 

  * To check the Android version number, click on this link <a href="https://familytime.io/kb/general/determining-android-os.html#:~:text=device's%20operating%20system.-,1,current%20operating%20system.">here</a>
  
  * To turn on developer, click on this link <a href="https://developer.android.com/studio/debug/dev-options#enable">here</a>

  2.) Clone the repository to your directory.
  
  2.) Locate and double click the run.bat file (Allow windows to run scripts if prompted).

  4.) Plug in the Android device into the computer. (You might get a notification on you phone to trust the computer for connection)

  5.) Check if Device information appears on the the form, if device is not recognized double check your connection, Android Version and/or if developer is enabled.

  6.) You are set to go
</p>
